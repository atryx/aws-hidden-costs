# Contributing

Thanks for contributing! This repo documents **real AWS billing surprises** — the costs that don't show up in pricing calculators.

## How to Contribute

1. Fork this repo
2. Add your hidden cost under the appropriate section (or create a new one)
3. Follow the format:
   - **Clear title** with the service/feature name
   - **The surprise** — what happened
   - **Why it costs so much** — the pricing mechanic
   - **How to avoid it** — actionable fix
   - **Estimated monthly impact** — ballpark figures where possible
4. Submit a PR

## Guidelines

- Include real dollar amounts or per-unit pricing where possible
- Focus on costs that surprise people, not well-known pricing
- Be specific about regions/services where pricing differs
- No vendor bashing — focus on helping people save money
